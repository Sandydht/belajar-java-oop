# Membuat Object
- Object adalah hasil instansiasi dari sebuah class.
- Untuk membuat object kita bisa menggunakan kata kunci ``` new ```, dan diikuti dengan nama class dan kurung ``` () ```
- Kode: Object
```java
var person1 = new Person();
Person person2 = new Person();

Person person3;
person3 = new Person();
```
